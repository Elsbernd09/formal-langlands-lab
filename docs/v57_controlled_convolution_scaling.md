# V57 Phase 143: Controlled Convolution Scaling

Version 57 proves scaling for the explicit controlled convolution prototypes at `n = 6` and `n = 12`.

## Motivation

Versions 55 and 56 proved additivity for the explicit controlled convolution prototypes.

Version 57 proves scaling, completing explicit bilinearity for both prototypes.

## Main Results

For `n = 6`:

`C₆(c • f, g) = c * C₆(f, g)`

and:

`C₆(f, c • g) = c * C₆(f, g)`

For `n = 12`:

`C₁₂(c • f, g) = c * C₁₂(f, g)`

and:

`C₁₂(f, c • g) = c * C₁₂(f, g)`

## Why This Matters

Dirichlet convolution is bilinear.

This phase completes the verified explicit bilinearity story for two controlled convolution prototypes.

## Honest Scope

These are still explicit prototype theorems for `n = 6` and `n = 12`, not yet a general theorem for all `n`.
