# Formal Examples Library

This phase adds a formal examples library to Version 2 of the Formal Langlands Laboratory.

## Motivation

The project now contains a stronger arithmetic foundation: verified divisor sums, classical arithmetic functions, finite multiplicativity tests, and Euler product approximations.

This examples layer collects those results into a clean showcase.

## Arithmetic Examples

The arithmetic examples file verifies concrete computations for:

- divisor-counting function
- sum-of-divisors function
- square-divisor-sum function
- multiplicativity tests
- two-prime Euler product approximations

## L-series Examples

The L-series examples file packages classical arithmetic functions into formal Dirichlet series.

It includes formal L-series for:

- divisor-counting coefficients
- sum-of-divisors coefficients
- square-divisor-sum coefficients

## Verified Computations

The project verifies examples such as:

- divisor count at 6 equals 4
- sum of divisors at 6 equals 12
- square divisor sum at 6 equals 50
- divisor-count Euler product approximation equals 36
- sum-of-divisors Euler product approximation equals 198
- square-divisor-sum Euler product approximation equals 2754

## Importance

This phase makes the project much easier to explain.

Instead of only having abstract infrastructure, the repository now includes a verified examples layer showing how the formal machinery computes recognizable number-theoretic objects.
