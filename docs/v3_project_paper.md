# Version 3 Project Paper

The main Version 3 project paper is located at:

`paper/formal_langlands_laboratory_v3.md`

This paper explains:

- the Version 1 architecture
- the Version 2 arithmetic upgrade
- the Version 3 rigorous arithmetic core
- gcd and coprimality
- coprime multiplicativity
- prime-power arithmetic tables
- Euler coefficient comparison
- theorem indexing
- module classification
- results dashboard
- limitations and future work

The paper is intended to present the project as a serious Lean 4 formalization laboratory with an honest separation between rigorous arithmetic modules and toy Langlands-inspired prototype modules.
