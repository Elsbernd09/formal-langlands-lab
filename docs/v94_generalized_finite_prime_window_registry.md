# V94 Phase 180: Generalized Finite Prime-Window Registry

Version 94 packages the generalized finite prime-window arc.

## Motivation

Versions 91 through 93 moved the project beyond hardcoded p = 2 and p = 3 local factors.

Version 94 packages this generalized work into a registry.

## Included Layers

Version 94 packages:

- generalized single prime-power windows
- length-four and length-three coefficient windows
- generalized two-prime product support windows
- expanded term lists
- combined coefficient lists
- complete-multiplicativity matching
- collision predicates
- p = 2, q = 3 collision-free recovery
- degenerate collision examples

## Main Package

The main package is:

`v94_generalized_finite_prime_window_registry_package`

It records a tracked result count of 21.

## Why This Matters

The project now has a reusable finite prime-window framework.

The earlier p = 2 and p = 3 Euler-product prototype becomes a special case of a more general finite two-parameter construction.

## Honest Scope

This remains finite and controlled.

Version 94 does not claim infinite Euler products, analytic convergence, or general all-prime Euler products.

It also does not claim collision-freeness for every natural-number pair p and q. V93 already records degenerate collision examples.
