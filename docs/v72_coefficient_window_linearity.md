# V72 Phase 158: Coefficient-Window Additivity and Scaling

Version 72 proves linearity for controlled Dirichlet-series coefficient aggregates.

## Motivation

Version 71 defined coefficient windows for arithmetic functions over finite index windows.

Version 72 proves that the finite coefficient aggregates behave linearly over arithmetic functions.

## Main Results

For the window `n ≤ 6`:

`A₆(f + g) = A₆(f) + A₆(g)`

and:

`A₆(c • f) = c * A₆(f)`

For the window `n ≤ 12`:

`A₁₂(f + g) = A₁₂(f) + A₁₂(g)`

and:

`A₁₂(c • f) = c * A₁₂(f)`

## Why This Matters

Dirichlet series are linear in their coefficients.

This phase proves the finite controlled version of that principle over the project's arithmetic-function layer.
