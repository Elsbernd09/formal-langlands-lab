# Project Paper

The main Version 2 project paper is located at:

`paper/formal_langlands_laboratory_v2.md`

This paper explains:

- the motivation for the project
- the Version 1 architecture
- the Version 2 arithmetic upgrade
- verified divisor sums
- classical arithmetic functions
- multiplicativity testing
- Euler product verification
- formal examples
- current limitations
- future work

The paper is intended to help readers understand the project as a serious Lean 4 formal verification prototype rather than only a collection of source files.
