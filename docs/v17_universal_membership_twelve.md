# Version 17 Universal Membership Agreement for 12

Version 17 proves the full universal membership agreement theorem for 12.

`d ∈ v6DivisorFinset 12 ↔ d ∈ v10ExpectedDivisorsTwelve`

### Bounded Region

### Out-of-Range Region

For values greater than 12, it uses the general out-of-range agreement theorem from Version 14.
