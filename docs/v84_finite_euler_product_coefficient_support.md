# V84 Phase 170: Finite Euler-Product Coefficient Support Registry

Version 84 packages the finite coefficient support used by the controlled two-factor Euler-product prototype.

## Motivation

Versions 81 through 83 proved expanded-term matching, complete-multiplicativity coefficient matching, and finite aggregate/product identity.

Version 84 packages the exact finite support underlying those results.

## Main Support

The controlled two-factor Euler-product support has sixteen combined indices:

`[1, 3, 9, 27, 2, 6, 18, 54, 4, 12, 36, 108, 8, 24, 72, 216]`

These arise from multiplying the finite prime-power windows:

- p = 2 window: `[1, 2, 4, 8]`
- p = 3 window: `[1, 3, 9, 27]`

## Main Results

Version 84 records:

- finite support length equals 16
- expanded local product terms
- combined coefficient terms
- complete-multiplicativity support matching
- aggregate matching
- product-identity registry for identity, square, and cube functions

## Honest Scope

This phase is finite and controlled.

It does not claim an infinite Euler product.

It gives the project a clean coefficient-support registry for future finite coefficient matching.
